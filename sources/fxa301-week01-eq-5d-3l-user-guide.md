## EQ-5D-3L User Guide

The document provides an overview of the EQ-5D-3L, a widely used tool for measuring health-related quality of life. It outlines instructions on how to use the instrument, including self-completion and proxy versions suitable for various populations. The guide explains that the EQ-5D-3L comprises two parts: a descriptive system with five dimensions (mobility, self-care, usual activities, pain/discomfort, anxiety/depression) each with three levels of severity, and an EQ VAS scale ranging from 0 to 100 where higher scores indicate better health. The manual also details the scoring of these scales and how to organize data collected through this instrument for clinical trials, population studies, and other research contexts. It includes guidance on obtaining translations and modes of administration, as well as references to additional resources available online through the EuroQol website.
