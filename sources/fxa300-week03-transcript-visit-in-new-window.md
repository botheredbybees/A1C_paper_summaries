## Breaking Down Short-Term Memory with Baddeley and Hitch's Working Memory Model

This transcript explains the working memory model proposed by Baddeley and Hitch to redefine short-term memory as a multifaceted system rather than a unified concept. The model includes four components: the central executive, phonological loop, visual-spatial sketchpad, and episodic buffer. Each component has specific roles such as managing auditory information for the phonological loop or handling spatial and visual data for the visual-spatial sketchpad. The theory is supported by empirical evidence showing differences in task performance when using different processing systems compared to the same system.
