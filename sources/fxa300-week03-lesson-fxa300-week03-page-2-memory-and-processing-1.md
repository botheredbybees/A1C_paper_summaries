## Memory and Processing

This lesson explores the fundamentals of memory formation, loss, and auditory processing through cognition. It covers echoic memory, working memory components (phonological loop, visuospatial sketchpad, episodic buffer, central executive), long-term memory storage, and the role of rehearsal in transferring information from short-term to long-term memory. The lesson includes discussions on musical earworms, involuntary musical imagery (INMI), and mechanisms such as the phonological loop in working memory involved in storing music-related information. It also touches upon attentional blindness and deafness, emphasizing how selective attention influences perception.
