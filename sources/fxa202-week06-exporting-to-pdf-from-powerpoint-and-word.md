## Exporting to PDF from PowerPoint and Word

This document provides instructions on how to export documents created in Microsoft PowerPoint and Word into a smaller, more manageable PDF format. For PowerPoint users, the process involves going to File - Export and selecting the desired file name and location for the exported PDF. Similarly, for Word users, it is necessary to select File - Save As and choose PDF as the export format under 'Export Formats.' The document emphasizes that exporting files into PDF reduces their size, making them easier to share and submit via platforms like MyLO assessment dropboxes without overloading internet connections.
