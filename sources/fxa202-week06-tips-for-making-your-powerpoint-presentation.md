## Tips for making your PowerPoint presentation

This document provides practical tips for creating a PowerPoint presentation. It begins by instructing users to open Microsoft PowerPoint and create a blank presentation. The first slide should include the presenter's full name in the heading text box. Users are guided on how to add new slides with various layouts or as a blank slide. For including images, there are instructions on inserting pictures from files using the Insert tab, as well as an alternative method of dragging and dropping photos directly into PowerPoint from file explorers like Finder (Mac) or File Explorer (Windows). The document explains how to resize and reposition these images. Text boxes can be added by selecting the Insert tab, clicking Text Box, choosing a location on the slide, typing in text, and deleting them if necessary by hitting the delete button after clicking their borders or photos respectively. Finally, it covers saving the presentation using the save icon on the top menu bar.
