## Value Sets for EQ-5D-5L

This book offers a comprehensive overview and detailed examination of the value sets developed for the EQ-5D-5L, an updated version of the widely used health-related quality-of-life instrument. The primary editors are Nancy Devlin, Bram Roudijk, and Kristina Ludwig. The work aims to assist users in understanding the attributes and limitations of these value sets, which are essential for economic evaluations and health technology assessments (HTAs). A significant focus is on the transition from the original 3L version to the new 5L version, highlighting improvements such as more gradual changes between health states and methods used to develop value sets for each jurisdiction. The book provides a detailed country-by-country overview of methodologies employed in generating these value sets along with comparative analyses that help users decide which set is most appropriate for their context. It concludes by addressing future directions, including ongoing debates about the sensitivity of different EQ-5D versions and conditions under which developing new value sets might be warranted.
