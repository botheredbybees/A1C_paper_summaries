## Creating Resources - Memory Trees and Boxes

The document discusses the creation of memory trees and boxes as tools for engaging people living with dementia in meaningful conversations. Memory trees involve a printout or drawing of a tree to which individuals can add pictures or words that trigger memories, illustrating personal identity, passions, and culture. This process encourages dialogue about cherished items and stories not previously shared. The finished memory tree serves as a resource for initiating ongoing discussions.

Memory boxes are physical containers where family members, carers, or friends gather items from the person's past that hold significant meaning. These items can include photographs, toys, sports memorabilia, and music recordings, among others, which are stored to evoke memories through sensory experiences. Both memory trees and boxes foster engagement and connection between individuals living with dementia and their loved ones.

The document emphasizes the importance of allowing those with dementia to share their world view and memories in a way that is meaningful and engaging for both parties.
