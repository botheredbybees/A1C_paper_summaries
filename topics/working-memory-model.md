## Working Memory Model

The Working Memory Model proposed by Baddeley and Hitch is a theoretical framework that describes how the brain processes and temporarily stores information. It challenges earlier notions of short-term memory by introducing distinct components such as the Central Executive, Phonological Loop, Visual Spatial Sketchpad, and Episodic Buffer. The model emphasizes the active manipulation of information through these specialized subsystems rather than passive storage alone. Echoic memory and implicit/explicit distinctions also play significant roles in understanding how sensory input transitions into more stable forms of memory. These elements collectively suggest a dynamic system capable of managing multiple cognitive tasks simultaneously, providing insights into both normal functioning and disorders affecting specific aspects of working memory.

## Key Findings

- The Working Memory Model includes distinct components like the Central Executive, Phonological Loop, Visual Spatial Sketchpad, and Episodic Buffer.
- These components work together to actively manipulate information rather than simply storing it passively.
- Echoic memory plays a role in how auditory input is processed before becoming part of working or long-term memory.
