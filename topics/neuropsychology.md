## Neuropsychological Assessment in Dementia Care

The documents provided highlight key aspects of neuropsychological assessment and care for individuals with dementia. The bio-psycho-social model presented emphasizes the importance of a holistic approach to managing dementia, integrating biological, psychological, and social factors. This model advocates for an understanding that cognitive deficits in patients with dementia are not solely due to neurodegenerative processes but also influenced by emotional states and social interactions. In contrast, the Mini-Mental State (MMS) assessment offers a practical method for clinicians to evaluate cognitive functions such as memory, language, attention, and problem-solving skills. Despite these complementary approaches, there is a noted discrepancy in addressing explicit versus implicit memory systems, where MMS primarily focuses on explicit memory but may not sufficiently assess implicit memory deficits, which are critical for understanding functional capabilities of dementia patients. This gap underscores the need for comprehensive neuropsychological assessments that can capture both explicit and implicit memory functions.

## Key Findings

- Bio-psycho-social model is essential in managing dementia care.
- Mini-Mental State assessment is a practical tool for clinicians to assess cognitive functions.
- There is a focus on explicit memory within MMS, but it lacks evaluation of implicit memory.

## Open Questions

- How can neuropsychological assessments be improved to address both explicit and implicit memory deficits in dementia patients?
