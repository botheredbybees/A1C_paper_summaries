## Implicit Memory

Sources discuss the distinction between implicit and explicit memory systems. Both sources highlight that implicit memory operates unconsciously to facilitate performance of tasks through procedural knowledge and priming effects, without the need for conscious recollection. Source 2 introduces Baddeley and Hitch's model which elaborates on working memory components including the phonological loop and visuospatial sketchpad, indicating how these relate to both explicit and implicit memory processes but with a stronger focus on implicit memory's role in automaticity and habit formation. However, there is an underexplored tension between the detailed workings of these memory systems within musical contexts as both sources do not fully integrate the specifics of music-induced implicit memories.

## Key Findings

- Implicit memory operates without conscious awareness to facilitate task performance through procedural knowledge and priming effects.
- Baddeley and Hitch's model of working memory encompasses elements relevant to both explicit and implicit memory, highlighting their interrelations.
- The role of implicit memory in habit formation is a fundamental aspect of its function.
